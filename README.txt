#  AI-Powered E-Commerce Analytics & Recommendation System

##  Project Overview

This project analyzes e-commerce transactional data to extract meaningful insights and build intelligent systems for customer segmentation and product recommendation. It combines data analysis, SQL, machine learning, and data visualization.

---

##  Features

*  Data Cleaning & Preprocessing using Python
*  Exploratory Data Analysis (EDA)
*  MySQL Database Integration
*  Customer Segmentation using K-Means Clustering
*  Recommendation System using Cosine Similarity
*  Interactive Dashboard using Power BI

---

##  Technologies Used

* **Python** (Pandas, NumPy, Matplotlib, Scikit-learn)
* **SQL (MySQL)**
* **Power BI**
* **Jupyter Notebook / VS Code**

##  Key Insights

* Top products contribute significantly to overall revenue
* A small number of customers generate most of the sales (Pareto Principle)
* Sales show clear seasonal trends
* High-selling products are not always the highest revenue generators

---

##  Machine Learning

Used **K-Means Clustering** to segment customers into:

* High Spenders 
* Medium Spenders 
* Low Spenders 

---

##  Recommendation System

Implemented using **Cosine Similarity**:

* Identifies similar customers
* Suggests products based on purchase behavior

##  How to Run the Project

1. Clone the repository
2. Install required libraries:

```
pip install pandas numpy matplotlib scikit-learn mysql-connector-python
```

3. Run the Jupyter Notebook step by step
4. Open Power BI file (`dashboard.pbix`) to view dashboard
